# Auto Encoder

A small PyTorch project that trains a convolutional autoencoder on the MNIST dataset.

The notebook loads MNIST images, trains an encoder-decoder model to reconstruct the digits, plots the training loss, and displays original images beside their reconstructions.

## Files

- `Auto_Encoder.ipynb` - main training and visualization notebook

## Requirements

- Python
- PyTorch
- torchvision
- matplotlib
- numpy
- tqdm
